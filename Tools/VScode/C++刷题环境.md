# **安装MinGW**

推荐官网:[MinGW-w64 - for 32 and 64 bit Windows - Browse Files at SourceForge.net](https://sourceforge.net/projects/mingw-w64/files/)

一步到位:https://zjnu.qingyandark.top:88/userfiles/programs/mingw64_12.2.0.zip

下载好之后 右键压缩包`全部解压缩`到一个路径`没有中文`且`容易管理`的地方

添加环境变量:
![[image/MinGW环境变量.png]]
